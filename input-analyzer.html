Input detail

Each column is from the latest 30 s row in D-input-processing.tsv for that input. The underlying metric is monitoring.dataset.<input_id>.processing_time.histogram in the diagnostic NDJSON (extractor converts ns → ms). That histogram is a lifetime reservoir, not a per-window delta: mean / median / percentiles describe how long codec + input-side processing took for observations accumulated in the histogram since the input started.
